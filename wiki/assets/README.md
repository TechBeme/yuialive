# Wiki Assets

This folder contains images, diagrams, and other assets used in the wiki documentation.

## Structure

```
assets/
├── payment/          # Payment integration screenshots
├── streaming/        # Streaming API diagrams
├── auth/            # Authentication flow diagrams
└── deployment/      # Deployment guide images
```

## Guidelines

- Use descriptive filenames (e.g., `payment-flow-diagram.png`)
- Optimize images for web (compress before uploading)
- Prefer PNG for screenshots, SVG for diagrams
- Maximum image size: 2MB per file
